#3: Registration problems

1. To create an account you need to use an existing email address and a unique nickname.

2. To register go on registration page by clicking the "Create an account" button or pressing the "Log in" button in the upper right corner of the menu line. Fill in all the fields in the form and press the "Register" button.

3. If all the data is correct, an account will be created. Otherwise, you will get an error message.

- A user with that username already exists means that the username you've entered has been alredy reserved. Choose another one.

- The two password fields didn't match. means that the passwords you've entered do not coincide. Try to fill in the password fields again they must match.

- The two password fields didn't match means that the passwords you've entered do not coincide. Try to fill in the password fields again they must match.

- Enter a valid email address means that the email address you've entered doesn't exist. Try choosing an existing address.

- User with this Email already exists means that somebody has already created an account with this email. Try choosing another existing address.

- This password is too short. It must contain at least 8 characters.

4. If you still have some questions, write on nerhneiro@gmail.com
